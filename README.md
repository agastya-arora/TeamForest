# TeamForest

## **`generate_list.py`**
Generates a random binary list representing the top-right block (of the adj. matrix of a bipartite graph) and appends the number of rows in that block.

## **`is_legal_list.py`**
Checks whether a list is legal, in other words: has the expected dimensions and contains only valid binary entries.

## **`list_to_matrix.py`**
Converts the list representation to a adjacency matrix. 

## **`matrix_to_list.py`**
Extracts the list representation from an adjacency matrix by using the top-right block. It does the opposite of what "list_to_matrix.py" is doing.

## **`get_all_vertex_subsets_of_size_i.py`**
Returns every possible vertex subset of a specified size using combinations.

## **`get_all_minors.py`**
Generates the minors corresponding to all vertex subsets of a specified size.

## **`adjMatrix_to_adjList.py`**
A piece of code that turns an adjacency-matrix into an adjacency-list.

## **`detect_cycle.py/find_cycles.py`**
Uses depth-first search on an adjacency list to determine whether a graph contains a cycle.

## **`generate_examples.py`**
Generates random example graphs and runs the project’s checking procedure on each example.

## **`create_sequence.py`**
Creates a sequence by counting cycle-free vertex subsets of each size.

## **`submatrices.py`**
Extracts and displays consecutive square submatrices of a specified size.
